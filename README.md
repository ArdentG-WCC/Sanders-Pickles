#Sanders Pickles 

A responsive small-business website built for a local pickle company.

## Live Site
- https://ardentg-wcc.github.io/Sanders-Pickles/

## Built With
- HTML5
- CSS3
- Vanilla javaScript
- localStorage (cart persistence)
- GitHub Pages (deployment)

## Features
- Responsive product grid layout
- Dynamic cart with quantity selection
- Smart cart logic (prevents duplicate line spam)
- Persistent cart using localStorage
- Clear cart funcionality
- Animated checkout confirmation
- Styled order summary panel

What I Practiced
- DOM manipulation
- State management using arrays
- Syncing UI with application state
- Persisting state in localStorage
- Debugging JavaScript errors
- Deploying production-ready code
